# 📝 Personal Todo List App with User Authentication

A fully functional, clean, and responsive **Todo (Task Management) Web Application** built using **Django** and styled with modern CSS. The application features a secure user authentication system, allowing individuals to register, log in, and manage their personal tasks independently.

🔗 **Live Demo:** [https://todo-djangotest.onrender.com/](https://todo-djangotest.onrender.com/)

---

## ✨ Features

* **User Authentication:** Secure Signup, Login, and Signout functionality using Django's built-in authentication system.
* **Session-based Routing:** Unregistered or logged-out users are automatically blocked from accessing task lists.
* **Full CRUD Functionality:** Users can Create, Read, Update (Edit), and Delete tasks safely.
* **Interactive UI Utilities:**
    * Automatic form resetting on validation or registration errors.
    * Error messages that cleanly fade/disappear after 10 seconds using native JavaScript.
    * JavaScript confirmation popup dialogs to prevent accidental task deletions.
* **Responsive Styling:** Clean layout with custom interactive buttons, beautiful layout gradients, and absolute positioning for top-level navigation.

---

## 🛠️ Tech Stack

* **Backend:** Python, Django Framework
* **Frontend:** HTML5, CSS3 (Flexbox, Absolute Positioning), JavaScript (ES6+)
* **Database:** SQLite (Default development database)
* **Production Server & Hosting:** Gunicorn, WhiteNoise (Static file management), Render Cloud Platform
